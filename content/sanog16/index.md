<td><p>SANOG XVI<br />
15-23 July, 2010 Paro, Bhutan</p>
<p>The venue for SANOG XVI is Paro, Bhutan. The event will be hosted by Bhutan Telecom.<br />
</p>
<p>June 25:2010: <a href="program.htm">SANOG XVI Draft Program</a></p>
<p>June 25, 2010 : <a href="fellowship.htm">SANOG XVI Fellowship Results</a></p>
<p>May 30, 2010: SANOG XVI <a href="registration.htm">Registration Opens</a></p>
<p>May 30, 2010: SANOG XVI <a href="program.htm">Draft Program</a></p>
<p>April 20, 2010 : SANOG XVI <a href="cfp.htm">Call for Papers</a></p>
<p>April 15, 2010 : SANOG XVI <a href="fellowship.htm">Fellowship Announced</a></p>
<p>April 6, 2010 : SANOG XVI website udpated</p>
<p><strong>Keynote Speaker : "Internet Operations" by Yoshinobu Matsuzaki (Maz), IIJ, Japan:</strong> <strong></strong> Yoshinobu Matsuzaki is a Senior Engineer at Internet Initiative Japan Inc. (IIJ/AS2497), a pioneering commercial ISP in Japan. He has a decade of network operation experience with the IIJ backbone network team. His areas of expertise include network design, network operation, network security, and DNS. He has given numerous talks at JANOG, NANOG, RIPE, and APNIC meetings, as wells as other workshops.</p>
<p><strong>About</strong> <strong>Bhutan Telecom</strong> : Bhutan Telecom is the incumbent telecom service provider in Kingdom of Bhutan. It provides fixed line telephony, mobile services and Internet services. This is the second time Bhutan Telecom is hosting SANOG. Previously it hosted SANOG in 2005.</p>
